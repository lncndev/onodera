# Onodera
Anime and manga database

## Installation
You'll have to build it yourself as it's still in development.

 1) `yay -Sy redis`
 2) `git clone https://github.com/onoderproject/onodera`
 3) `cd onodera`
 4) `crystal build src/onodera.cr`
 5) `./onodera`
